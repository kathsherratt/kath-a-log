## The uncertainty/VOI framework

Two axes, expressed as **value of information**:
- **x-axis**: value of additional epidemiological information (what's happening)
- **y-axis**: value of additional public health information (what to do)

Current plot:

```
PH uncertainty / VOI
(what to do)
    │
H   │  AMR             Novel pandemic
    │                  COVID wk1
    │
    │         ←── outbreak evolves ───
    │
    │  Seasonal flu    Cholera/conflict
    │  Endemic malaria Famine
L   │                  COVID wk52?
    └──────────────────────────────────── Epi uncertainty / VOI
       L                              H   (what's happening)
```

Key insight: axes are continuous not categorical; outbreaks move through the space over time.

Nowcasting = high y (informs response), low x (doesn't add to epi dynamics understanding).
